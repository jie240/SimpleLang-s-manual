SimpLang 教学文档

欢迎！SimpLang 是一门语法极简、动态类型的解释型语言。如果你有 C 语言基础，半小时即可掌握。本文档将带你从零开始，逐步掌握 SimpLang 的所有特性。

📥 安装与运行

SimpLang 提供两种运行方式：

  • 图形化 IDE：下载 SimpleLangIDE.exe，双击运行，即可编写和运行代码。
  • 命令行解释器：使用 Python 运行解释器脚本（需要 Python 3.6+）。

推荐初学者使用 IDE，它内置了语法高亮、一键运行和打包功能。

👋 第一个程序：Hello, World

打开 IDE，在编辑区输入以下代码，按 F5 运行：

print "Hello, World!";

输出区将显示 Hello, World!。恭喜你，已经成功运行了第一个 SimpLang 程序！

📌 变量与赋值

使用 var 关键字声明变量。变量名由字母、数字和下划线组成，不能以数字开头。

var age = 25;
var name = "Alice";
var flag = true;      // Boolean: 0 is false, non-zero is true
var x;                // Uninitialized, defaults to 0

已声明的变量可以直接赋值：

age = 26;
name = "Bob";
💡 提示：SimpLang 是动态类型的，变量的类型由赋值决定，可以随时改变类型。

🔢 数据类型

SimpLang 支持以下基本类型：

类型示例说明
数字10, 3.14, -5整数和浮点数统一为数字类型
字符串"hello"用双引号包围，支持转义 \" 和 \n
布尔true / false用数字表示，条件判断中 0 为假，非0为真
数组var arr[10];固定大小的一维数组，元素默认为 0

➕ 运算符

SimpLang 支持常见的算术、比较和逻辑运算符。

算术运算符

运算符含义示例
+加法 / 字符串连接3 + 5 → 8；"a" + "b" → "ab"
-减法10 - 4 → 6
*乘法2 * 3 → 6
/除法7 / 2 → 3.5
%取模7 % 2 → 1

比较运算符

==等于!=不等于
<小于>大于
<=小于等于>=大于等于

逻辑运算符

&&逻辑与||逻辑或!逻辑非
var a = 5;
var b = 10;
if (a > 0 && b < 20) {
    print "Both conditions are true";
}

💬 注释

单行注释使用 //，多行注释使用 /* ... */。

// This is a single-line comment
var x = 10;   // Can also be placed after a statement

/*
   This is a
   multi-line comment
*/

🚦 if-else 条件语句

var score = 85;
if (score >= 90) {
    print "Excellent";
} else if (score >= 60) {
    print "Pass";
} else {
    print "Fail";
}

条件表达式必须用圆括号 ( ) 包围，执行体用花括号 { }。

🔄 while 循环

var i = 0;
while (i < 5) {
    print "i = ", i, "\n";
    i = i + 1;
}

🧩 函数定义与调用

使用 func 关键字定义函数。函数可以接受多个参数，并支持递归。

func add(a, b) {
    return a + b;
}

var result = add(3, 5);
print result;   // Output 8

返回值

使用 return 语句返回值。如果没有 return，函数默认返回 0。

递归

函数可以直接或间接调用自身。例如计算阶乘：

func factorial(n) {
    if (n <= 1) { return 1; }
    return n * factorial(n - 1);
}

🗂️ 数组

声明数组时需要指定大小：

var numbers[5];       // Create array of length 5, elements default to 0
numbers[0] = 10;
numbers[1] = 20;

访问数组元素：

print numbers[0];      // 10
var idx = 1;
print numbers[idx];    // 20
⚠️ 注意：数组索引从 0 开始，访问越界会导致运行时错误。

📝 字符串

字符串可以用 + 进行拼接：

var greeting = "Hello, " + "world!";
print greeting;

字符串可以包含转义字符：

print "First line\nSecond line";
print "She said: \"Hello!\"";

⚙️ 内置函数

len(str)

返回字符串的长度。

var s = "SimpLang";
print len(s);   // 8

to_int(str) / to_str(num)

类型转换函数。

var numStr = "42";
var num = to_int(numStr);   // 42
var str = to_str(100);      // "100"

🖥️ 输入与输出

print

print 可以输出多个值，用逗号分隔，它们会被直接拼接输出（无自动空格）。

print "x = ", x, "\n";

input 和 input_str

input 读取数字，input_str 读取字符串。在 IDE 中会弹出输入对话框。

var age;
print "Please enter age: ";
input age;
print "Your age is: ", age;

var name;
print "Please enter name: ";
input_str name;
print "Hello, ", name;

📦 打包为 EXE

在 SimpLang IDE 中，点击菜单 运行 → 打包为 EXE...，即可将当前脚本编译成独立的 Windows 可执行文件。生成的 EXE 无需安装任何依赖，可直接运行。
